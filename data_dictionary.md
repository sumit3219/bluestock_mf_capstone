# Data Dictionary

## fact_nav

- amfi_code : Unique mutual fund code
- date : NAV date
- nav : Net Asset Value

## fact_transactions

- transaction_type : Type of transaction
- amount : Transaction amount
- date : Transaction date

## fact_performance

- return_1yr_pct : 1 year return percentage
- return_3yr_pct : 3 year return percentage
- expense_ratio_pct : Expense ratio of fund
- risk_grade : Risk category
